# Автоматизированная система управления рулонным пресс-подборщиком
Промышленная система управления (контроллер) для тракторных пресс-подборщиков рулонного типа (Pottinger 3120, 3200, 3500 и аналоги). 
Система автоматизирует процесс обвязки тюка (сеткой или шпагатом), подает звуковые и визуальные сигналы оператору и ведет энергонезависимый учет количества собранных рулонов.
## 🏗 Архитектура системы
Проект построен по топологии **Master-Slave (Ведущий-Ведомый)** с использованием надежной промышленной шины передачи данных **RS485**.
1. **Блок Пульта (Master)** — устанавливается в кабине трактора. Отвечает за интерфейс оператора (LCD экран), настройку таймеров, звуковое оповещение внутри кабины и передачу команд.
2. **Блок Пресса (Slave)** — устанавливается в пыле-влагозащищенном коробе на самом прессе. Считывает концевые датчики, управляет силовыми реле (электромотор, маяк, штатный гудок пресса) и хранит счетчики во внешней EEPROM-памяти.
## ✨ Основные возможности
* **Два режима работы:** Автоматический (по датчикам) и Ручной (принудительный запуск мотора с пульта).
* **Два материала обвязки:** Сетка и Шпагат (с независимыми таймерами работы мотора).
* **Маршрутизация звука:** Возможность выбора, где будет звучать сигнал — только в кабине, только на прессе, или везде одновременно.
* **Сложные звуковые паттерны:** Разные фазы работы звучат по-разному, чтобы оператор мог управлять трактором вслепую (на слух).
* **Антизависание шины I2C:** Внедрен программный Timeout (25 мс) и автоматический сброс шины I2C для защиты от электромагнитных помех (генератор, стартер, силовые реле).
* **Отказоустойчивость:** При обрыве кабеля RS485 пресс продолжает работу в автономном режиме и принудительно включает внешний гудок для оповещения. Максимальное время работы моторов программно ограничено (защита от возгорания при заклинивании механики).
* **Энергонезависимая память:** Хранение настроек и счетчика тюков на внешнем чипе I2C EEPROM (AT24Cxx).

## 🔌 Распиновка (Hardware Pinout)
### Блок Пульта (Master - Arduino Nano/Uno)

| Пин | Устройство | Тип / Примечание |
| :--- | :--- | :--- |
| **D2** | Тумблер выбора материала (Сетка/Шпагат) | Замыкание на GND (INPUT_PULLUP) |
| **D3** | Кнопка Левая (Экран / Влево) | Замыкание на GND (INPUT_PULLUP) |
| **D4** | Тумблер режима (Ручной/Авто) | Замыкание на GND (INPUT_PULLUP) |
| **D11** | Кнопка Правая (Действие / Вправо) | Замыкание на GND (INPUT_PULLUP) |
| **D12** | Зуммер кабины | **Active-LOW** (Звук при LOW) |
| **D8** | RS485 RX (RO) | SoftwareSerial |
| **D9** | RS485 TX (DI) | SoftwareSerial |
| **D10** | RS485 EN (DE+RE) | Управление направлением передачи |
| **A4 (SDA)** | Дисплей LCD 1602 I2C | Линия данных |
| **A5 (SCL)** | Дисплей LCD 1602 I2C | Линия тактирования |

### Блок Пресса (Slave - Arduino Nano/Uno)

| Пин | Устройство | Тип / Примечание |
| :--- | :--- | :--- |
| **D2** | Датчик плотности (Старт цикла) | Замыкание на GND |
| **D3** | Датчик старта обвязки | Замыкание на GND |
| **D4** | Датчик конца обвязки (Нож) | Замыкание на GND |
| **D5** | Датчик открытия двери (Выгрузка) | Замыкание на GND (Нормально замкнут) |
| **D8** | Кнопка Reset (на корпусе пресса) | Замыкание на GND |
| **D11** | Аппаратный тумблер (Сетка/Шпагат) | Резервное управление без пульта |
| **D6** | Реле мотора (Шпагат) | +5V (Active-HIGH) |
| **D10** | Реле мотора (Сетка) | +5V (Active-HIGH) |
| **D7** | Реле гудка пресса | +5V (Active-HIGH) |
| **D9** | Реле проблескового маячка | +5V (Active-HIGH) |
| **A0** | RS485 RX (RO) | SoftwareSerial |
| **A1** | RS485 TX (DI) | SoftwareSerial |
| **A2** | RS485 EN (DE+RE) | Управление направлением передачи |
| **A4 (SDA)** | Модуль I2C EEPROM AT24C | Линия данных (Адрес 0x50) |
| **A5 (SCL)** | Модуль I2C EEPROM AT24C | Линия тактирования |

## 🎵 Циклограмма и Звуковые сигналы
1. **Ожидание массы (NABOR MASSY)** — Тишина.
2. **Набрана плотность (STOP TRAKTOR!)** — Сработал датчик на D2. 
   * 🔊 Звук: **3 длинных гудка (по 400 мс)**.
3. **Начало обвязки (Вращение мотора)** — Отсчет времени `t_Stop` или старт вручную.
   * 🔊 Звук: **1 короткий (250 мс)**.
4. **Конец обвязки (OTKROY DVER!)** — Сработал датчик ножа на D4.
   * 🔊 Звук: **Спец-сигнал (Короткий 150мс + Длинный 800мс)**.
5. **Выгрузка завершена** — Датчик двери D5 разомкнулся и замкнулся. Счетчик `Total +1`.
   * 🔊 Звук: **2 коротких радостных (по 200 мс)**. Возврат к п.1.
6. **Авария (SBOY! PROVER!)** — Зависание механики, не сработал нож или мотор. 
   * 🔊 Звук: **10 быстрых тревожных гудков**.

## ⚙️ Управление и Настройки
### Вход в меню
* Переведите тумблер РЕЖИМ в положение **Авто [A]**.
* Зажмите обе кнопки (Левую и Правую) на **2 секунды**.
### Навигация
* **Левая кнопка** — перелистывание пунктов меню (Плотность, Остановка, Сетка, Шпагат, Звук).
* **Правая кнопка** — увеличение значения на +1.
* **Тумблер Сетка/Шпагат + Правая кнопка** — уменьшение значения на -1.
* **Сохранение и выход** — зажать обе кнопки на 2 секунды (на экране появится `SOHRANENO V PRES`).
### Режимы звука (Пункт меню `ZVUK`)
* `TOLKO PRES (0)` — Пищит только внешний автомобильный гудок на прессе.
* `TOLKO PULT (1)` — Внешний гудок отключен, работает только зуммер в кабине трактора.
* `PRES+PULT (2)` — Звук дублируется на оба блока синхронно.

## 🛠 Зависимости и Библиотеки (Для Arduino IDE)
Перед компиляцией убедитесь, что установлены следующие библиотеки:
* `Wire.h` (Встроена)
* `SoftwareSerial.h` (Встроена)
* `LiquidCrystal_I2C` (от Frank de Brabander или аналог)
**ВНИМАНИЕ РАЗРАБОТЧИКУ:** Аппаратный таймер Watchdog (`<avr/wdt.h>`) намеренно исключен из кода из-за конфликта со старыми Bootloader'ами на китайских платах Nano. Защита от зависаний реализована исключительно через программные таймауты и функцию `Wire.setWireTimeout(25000, true)`. При компиляции убедитесь, что используется актуальное ядро Arduino AVR Boards.
