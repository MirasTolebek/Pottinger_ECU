# Автоматизированная система управления рулонным пресс-подборщиком (АСУ-ПП)

Промышленный программно-аппаратный комплекс для полной автоматизации цикла обвязки (сеткой или шпагатом) рулонных сельскохозяйственных пресс-подборщиков (адаптировано под Pottinger, МТЗ и аналоги).

## 🏗 Архитектура системы
Система построена по отказоустойчивой топологии **Master-Slave (Ведущий-Ведомый)**. Связь между модулями осуществляется по дифференциальной промышленной шине **RS485** на скорости 9600 бод с контролем целостности пакетов через алгоритм контрольной суммы (XOR).

1. **Блок Пульта (Master)** — располагается в кабине трактора. Оснащен ЖК-дисплеем 1602 с I2C-интерфейсом, кнопками управления, тумблерами выбора режимов и внутренним зуммером оповещения. Выполняет функции центрального командования, конфигурирования таймеров и отображения телеметрии.
2. **Блок Пресса (Slave)** — монтируется в герметичном пыле-влагозащищенном шкафу непосредственно на раме пресс-подборщика. Опрашивает концевые выключатели и индуктивные датчики, управляет силовыми реле исполнительных механизмов (электромоторы обвязки, штатный звуковой сигнал, проблесковый маяк) и осуществляет циклическую запись статистики во внешнюю энергонезависимую память.

## ⚡ Ключевые особенности и защитные механизмы
* **Защита от электромагнитных помех (EMI):** Аппаратный таймаут шины I2C (`Wire.setWireTimeout(25000, true)`) на обоих блоках. При наводках от мощных двигателей или генератора трактора зависшая шина автоматически сбрасывается за 25 мс без остановки основного цикла программы.
* **Программный Сторожевой пес (Software Watchdog):** Реализован на аппаратном Таймере 1 блока Пульта. Автоматически перезагружает устройство (Soft Reset через `jmp 0` с предварительным `cli()`) при зависании дисплея или логики более чем на 4 секунды. Полностью исключает риск "вечной комы" микроконтроллера, свойственный стандартному аппаратному WDT на неоригинальных платах Arduino Nano.
* **Защита EEPROM от износа (Smart Wear Leveling):** Запись данных во внешнюю память AT24Cxx на блоке пресса переведена на неблокирующую функцию `updateEEPROM_Byte`. Запись байта физически происходит только в случае изменения его значения, что экономит время цикла (5 мс на байт) и продлевает ресурс чипа памяти до десятков лет интенсивной работы.
* **Аварийные таймеры электромоторов:** Время непрерывной работы приводов обвязки и ожидания датчика отсечки жестко ограничено настраиваемым параметром безопасности (5–30 сек). В случае заклинивания ножа или обрыва нити система мгновенно обесточит силовые реле, предотвращая перегрев двигателей и возгорание проводки.
* **Резервный автономный режим:** При полном обрыве кабеля связи RS485 блок пресса переходит в режим автономного автомата, принудительно перенаправляя все звуковые сигналы на внешний гудок, чтобы механизатор не пропустил момент формирования рулона.

## 🔌 Спецификация подключения (Hardware Pinout)

### Блок Пульта (Кабина - Master)
| Пин Arduino | Периферийное устройство | Тип сигнала / Логика работы |
| :--- | :--- | :--- |
| **D2** | Тумблер выбора материала (Сетка / Шпагат) | Вход, INPUT_PULLUP (GND = Сетка) |
| **D3** | Кнопка «ЭКРАН» (Левая) | Вход, INPUT_PULLUP (GND = Нажато) |
| **D4** | Тумблер режима работы (Ручной / Авто) | Вход, INPUT_PULLUP (GND = Ручной) |
| **D8** | Модуль RS485 — Вывод RO (Receive Output) | SoftwareSerial RX |
| **D9** | Модуль RS485 — Вывод DI (Data Input) | SoftwareSerial TX |
| **D10** | Модуль RS485 — Выводы RE + DE | Выход управления направлением (HIGH = Передача, LOW = Прием) |
| **D11** | Кнопка «ДЕЙСТВИЕ» (Правая) | Вход, INPUT_PULLUP (GND = Нажато) |
| **D12** | Пьезо-зуммер пульта в кабине | Выход, **Active-LOW** (Звук при LOW, тишина при HIGH) |
| **A4 (SDA)** | Дисплей LCD 1602 I2C | Шина данных I2C (Подтяжка к 5V) |
| **A5 (SCL)** | Дисплей LCD 1602 I2C | Шина тактирования I2C |

### Блок Пресса (Улица - Slave)
| Пин Arduino | Силовое устройство / Датчик | Направление / Логический уровень |
| :--- | :--- | :--- |
| **D2** | Датчик плотности рулона (Предельный прессователь) | Вход, INPUT_PULLUP (GND = Рулон сформирован) |
| **D3** | Датчик начала обвязки (Стартовый индуктивный) | Вход, INPUT_PULLUP (GND = Подача материала пошла) |
| **D4** | Датчик конца обвязки (Концевик ножа / отсечки) | Вход, INPUT_PULLUP (GND = Нож упал, обрезка выполнена) |
| **D5** | Датчик открытия задней двери камеры пресса | Вход, INPUT (HIGH = Дверь открыта, выгрузка) |
| **D6** | Реле управления электромотором шпагата | Выход, Active-HIGH (+5V = Включение силового пускателя) |
| **D7** | Реле управления штатным звуковым сигналом (Горн) | Выход, Active-HIGH (+5V = Звуковой сигнал включен) |
| **D8** | Тактовая кнопка «СБРОС / ТЕСТ» на корпусе блока | Вход, INPUT_PULLUP (GND = Нажато) |
| **D9** | Реле управления проблесковым маяком (Сигнальный свет) | Выход, Active-HIGH (+5V = Маяк включен) |
| **D10** | Реле управления электромотором сетки | Выход, Active-HIGH (+5V = Включение силового пускателя) |
| **D11** | Аварийный тумблер ручного выбора материала на прессе | Вход, INPUT_PULLUP (Используется при потере связи) |
| **A0** | Модуль RS485 — Вывод RO | SoftwareSerial RX |
| **A1** | Модуль RS485 — Вывод DI | SoftwareSerial TX |
| **A2** | Модуль RS485 — Выводы RE + DE | Выход направления передачи (HIGH = TX, LOW = RX) |
| **A4 (SDA)** | Чип памяти EEPROM AT24Cxx | Данные I2C (Физический адрес устройства: 0x50) |
| **A5 (SCL)** | Чип памяти EEPROM AT24Cxx | Такты I2C |
